📊 Sales Dashboard – Tableau Project

📌 Project Overview

This project presents an interactive Sales Dashboard built using Tableau to analyze business performance across multiple dimensions such as sales, profit, and product categories. The dashboard helps identify trends, compare performance, and support data-driven decision-making through clear visual storytelling.

🎯 Objectives:
• Analyze sales and profit trends over time
• Compare performance across categories and sub-categories
• Identify top-performing and low-performing products
• Provide interactive filtering for customized insights
• Demonstrate best practices in Tableau dashboard design

🛠️ Tools & Technologies:
• Tableau Desktop – Data visualization and dashboard creation
• Microsoft Excel / CSV – Data source preparation
• GitHub – Version control and project documentation

📊 Dashboard Features:
📈 Sales Trend Analysis – Visualizes sales over time
💰 Profit Analysis – Tracks profitability across segments
🧩 Category & Sub-Category Insights
🎯 KPI Indicators – Key performance metrics
🎛️ Interactive Filters – Region, Category, and Date
📌 Dynamic Visualizations – Bar charts, line charts, and maps

🚀 How to Use:
1. Download the .twbx file from this repository
2. Open it using Tableau Desktop
3. Explore the dashboard using available filters
4. Analyze trends and insights from the visualizations

📚 Learning Outcomes:

Through this project, the following skills were demonstrated:
• Data cleaning and preparation
• Dashboard design principles
• Interactive visualization techniques
• KPI development
• Data storytelling using Tableau
